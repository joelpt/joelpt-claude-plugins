# Tip of the Day - Claude Code Plugin

A Claude Code plugin that delivers a curated intelligence briefing at the start of each session. Tips are generated by Claude from your locally installed plugins, skills, and online sources like the Anthropic blog and Claude Code docs. The tip database self-refreshes every 24 hours.

## Installation

From the plugin marketplace:

```
/plugin marketplace add /path/to/totd-plugin
/plugin install tip-of-the-day@totd-dev
```

For local testing:

```
claude --plugin-dir /path/to/totd-plugin
```

## How It Works

- On **SessionStart**, the hook picks the highest-scored unshown tip from the database and displays it
- If all tips have been shown, the least-recently-shown tip is recycled
- **Tip generation** runs automatically in the background when unshown tips drop below a watermark (default: 5) or the database is stale (default: >24 hours)
- Generation discovers your local plugins/skills, builds a user profile from recent projects, then calls Claude CLI to produce new tips from local context and online sources
- A **curation pass** deduplicates tips, scores them for relevance, and retires stale entries

## On-Demand Usage

The `/totd` skill provides interactive access:

| Command | Description |
|---------|-------------|
| `/totd` | Show a tip |
| `/totd list` | Show recently seen tips |
| `/totd git` | Get a tip about a specific topic |
| `/totd stats` | View tip database statistics |
| `/totd regenerate` | Force tip generation now |

## Configuration

Edit `config/sources.json` to customize:

- **sources** - Add or remove online sources (blogs, docs, GitHub repos)
- **generation.low_watermark** - Minimum unshown tips before triggering generation (default: 5)
- **generation.staleness_hours** - Hours before database is considered stale (default: 24)
- **generation.max_tips** - Maximum tips retained in database (default: 200)
- **generation.target_unshown** - Target number of unshown tips per generation run (default: 30)

## Requirements

- **jq** - JSON processing (`brew install jq` / `apt install jq`)
- **claude** - Claude CLI (for tip generation)
